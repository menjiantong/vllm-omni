### 2.2 Single-Stream Block (OvisImageSingleTransformerBlock)

```
        Input: hidden_states, encoder_hidden_states, temb
                              │
                              ▼
              ┌───────────────────────────────┐
              │  Concat [Text, Image]         │
              │  hidden = [encoder_hid, hid]  │
              │  Shape: [B, txt_seq+img_seq, dim] │
              └───────────────┬───────────────┘
                              │
                              ▼
              ┌───────────────────────────────┐
              │     AdaLayerNormZeroSingle    │
              │                               │
              │  Output: norm_hidden, gate    │
              └───────────────┬───────────────┘
                              │
              ┌───────────────┴───────────────┐
              │                               │
              ▼                               ▼
    ┌─────────────────────┐       ┌─────────────────────┐
    │   Self Attention    │       │       MLP           │
    │                     │       │                     │

    │       │             │       │       │             │

    │       │             │       │       │             │

    │  SDPA (no causal)   │       │  mlp_hidden_states  │

    │  attn_output        │                 │
    └─────────┬───────────┘                 │
              │                             │
              └──────────────┬──────────────┘
                             │
                             ▼
              ┌───────────────────────────────┐
              │  Concat [attn_out, mlp_out]  │

## 3. Attention 机制详解

### 3.1 OvisImageAttention 数据流

```
        Input: hidden_states, encoder_hidden_states
                      │
        ┌─────────────┴─────────────┐
        │                           │
        ▼                           ▼
┌───────────────────┐     ┌───────────────────┐
│   Image QKV       │     │   Encoder QKV     │
│   to_qkv          │     │   add_kv_proj     │
│   [seq, dim]      │     │   [txt_seq, dim]  │
│       │           │     │       │           │
│       ▼           │     │       ▼           │
│  Split Q, K, V    │     │  Split Q, K, V    │
│  each [seq, dim]  │     │  each [txt_seq, dim]
│       │           │     │       │           │
│       ▼           │     │       ▼           │
│  Unflatten heads  │     │  Unflatten heads  │
│  [B,S,H,D/H]      │     │  [B,S,H,D/H]      │
│       │           │     │       │           │
│       ▼           │     │       ▼           │
│  RMSNorm(Q)       │     │  RMSNorm(Q)       │
│  RMSNorm(K)       │     │  RMSNorm(K)       │
└───────┬───────────┘     └───────┬───────────┘
        │                         │
        └────────────┬────────────┘
                     │
                     ▼
        ┌────────────────────────┐
        │  Concat [Encoder, Image] │
        │  Q: [B, txt+img, H, D/H] │
        │  K: [B, txt+img, H, D/H] │
        │  V: [B, txt+img, H, D/H] │
        └────────────┬───────────┘
                     │
                     ▼
        ┌────────────────────────┐
        │      Apply RoPE        │
        │                        │
        │  cos, sin from         │

## 4. 位置编码详解

### 4.1 3D RoPE 结构

```
        Position IDs (ids)
        Shape: [seq_len, 3]
        │
        │  ┌───────────────────────────────────────────┐
        │  │   ids format: [time, height, width]       │
        │  │                                           │
        │  │   Text tokens: [0, token_idx, token_idx]  │
        │  │   Image tokens: [0, h_idx, w_idx]         │
        │  └───────────────────────────────────────────┘
        │
        ▼
┌───────────────────────────────────────────────────────────────────┐
│                        OvisImagePosEmbed                          │
│                                                                   │
│   for axis in [time, height, width]:                             │
│       │                                                           │
│       ▼                                                           │
│   ┌─────────────────────────────────────────────────────────────┐ │
│   │              get_1d_rotary_pos_embed                         │ │
│   │                                                              │ │
│   │   freqs = 1.0 / (theta ** (torch.arange(0, dim, 2) / dim))  │ │
│   │   angles = pos * freqs                                       │ │
│   │   emb = torch.polar(torch.ones_like(angles), angles)        │ │
│   │                                                              │ │
│   │   Output: complex tensor [seq_len, dim/2]                   │ │
│   └─────────────────────────────────────────────────────────────┘ │
│       │                                                           │
│       ▼                                                           │
│   cos = emb.real  →  [seq_len, dim/2]                            │
│   sin = emb.imag  →  [seq_len, dim/2]                            │
│                                                                   │
│   Concat all axes:                                                │
│   cos_final = [cos_time, cos_h, cos_w] → [seq_len, 128]         │
│   sin_final = [sin_time, sin_h, sin_w] → [seq_len, 128]         │
│                                                                   │
│   axes_dim = [16, 56, 56]  →  total = 128 (head_dim)            │
└───────────────────────────────────────────────────────────────────┘
        │
        ▼
    Return: (cos, sin)
    Shape: [seq_len, 128] each
```

### 4.2 RoPE 应用过程

```
        Input: Q, K (after head split)
        Shape: [B, seq, H, D/H]
              │
              ▼
┌───────────────────────────────────────────────────────────┐
│                   RotaryEmbedding                          │
│                                                           │
│   For GPT-J style (interleaved=True):                    │
│                                                           │
│   Q_rotated = rotate_half(Q)                             │
│   Q_out = Q * cos - Q_rotated * sin                      │
│                                                           │
│   Where:                                                  │
│   rotate_half(x) = [-x[..., 1::2], x[..., ::2]]          │
│                                                           │
│   For GPT-NeoX style (interleaved=False):                │
│                                                           │
│   x1, x2 = x.chunk(2, dim=-1)                            │
│   rotate_half(x) = [-x2, x1]                             │
│   x_out = x * cos + rotate_half(x) * sin                 │
└───────────────────────────────────────────────────────────┘
              │
              ▼
    Output: Q_rope, K_rope
    Shape: [B, seq, H, D/H]
```

## 5. Latent Packing/Unpacking 可视化

### 5.1 Packing 过程

```
Original Latent: [B, 64, H/16, W/16]
Example: H=1024, W=1024 → [B, 64, 64, 64]

Step 1: Reshape
[B, 64, 64, 64] → [B, 64, 32, 2, 32, 2]
                 │     │  │  │  │  │
                 │     │  │  │  │  └─ 2x2 patch
                 │     │  │  │  └──── height/2
                 │     │  │  └─────── 2x2 patch
                 │     │  └────────── width/2
                 │     └───────────── channels

Step 2: Permute
[B, 64, 32, 2, 32, 2] → [B, 32, 32, 64, 2, 2]
                        │   │   │   │   │  │
                        │   │   │   │   │  └─ pack dim
                        │   │   │   │   └──── pack dim
                        │   │   │   └──────── channels
                        │   │   └──────────── width_idx
                        │   └──────────────── height_idx
                        └──────────────────── batch

Step 3: Reshape
[B, 32, 32, 64, 2, 2] → [B, 1024, 256]
                        │    │     │
                        │    │     └─ 64 * 2 * 2 = 256 features
                        │    └─────── 32 * 32 = 1024 tokens
                        └──────────── batch

Final: [B, 1024, 256]  (or [B, img_seq, 64] if using in_channels=16)
```

### 5.2 Unpacking 过程

```
Packed Latent: [B, 1024, 256]
              │
              ▼
Step 1: Reshape
[B, 1024, 256] → [B, 32, 32, 64, 2, 2]

Step 2: Permute
[B, 32, 32, 64, 2, 2] → [B, 64, 32, 2, 32, 2]

Step 3: Reshape
[B, 64, 32, 2, 32, 2] → [B, 64, 64, 64]
                        │    │   │   │
                        │    │   │   └─ W/16
                        │    │   └───── H/16
                        │    └───────── channels
                        └────────────── batch

Final: [B, 16, H/16, W/16] (for VAE decoder input)
```
